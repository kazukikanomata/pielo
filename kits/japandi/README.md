# ジャパンディ 世界観キット — JAPANDI Worldview Kit

**pielo worldview kit / v0.1 (prototype)**

「引き算の先に、豊かさがある。」——日本の侘び寂びと北欧のヒュッゲの合流点。
このキットは、あなたのプロダクトの見た目を**根拠のあるジャパンディ**にするための一式です。

## 中身

| ファイル | 用途 |
|---|---|
| `ai-style-prompt.md` | v0 / Lovable / Cursor / Claude Code に貼るスタイルプロンプト(日/英) |
| `theme.css` | CSS変数 + Tailwind v4 `@theme`。コピペで導入 |
| `tokens.json` | デザイントークン(Figma変数や他ツールへの取り込み用) |
| `fonts-and-type.md` | フォントペアと組み方の数値(字間・行間・ウェイト) |

## 使い方(3通り)

1. **AIに作らせる** — `ai-style-prompt.md` の内容を、UI生成の指示と一緒に貼る。出来上がりを本文末尾の「チェックリスト」で検品する
2. **既存プロジェクトに被せる** — `theme.css` を読み込み、色・角丸・影をトークン経由に置き換える
3. **デザインツールで使う** — `tokens.json` をFigmaの変数にインポートする

## なぜ、こう見えるのか(この様式の骨格)

- **余白**: 画面の半分以上をなにもない空間に。余白は「未使用領域」ではなく静けさという商品
- **色**: 生成り・砂・苔・炭。彩度を抑えた自然の色だけに絞ると、素材感が立ち上がる
- **文字**: 細い明朝/ライトウェイトのサンセリフを、広い字間でゆっくり読ませる
- **装飾**: 細い線と小さな図形だけ。塗りつぶさない・囲いすぎない

## 検品チェックリスト(AI出力がジャパンディになっているか)

- [ ] 画面の50%以上が余白か
- [ ] 使用色は低彩度のアースカラー+炭色のみか(原色・ネオンが混ざっていないか)
- [ ] 見出しは細いウェイトか(太字の主張がないか)
- [ ] ボタンが「塗りつぶした箱」ではなく、文字+下線 or 細枠になっているか
- [ ] 影・グラデーション・大きい角丸が排除されているか
