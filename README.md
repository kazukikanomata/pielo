# pielo

どんな世界観にも、化けられる。
デザイン史から生まれた6つの世界観を、UIで「試着」するギャラリーサイト。

## コンセプト

- **2つの玄関**
  - 問いから入る(コーゼーション): 3つの問いに答えると、近い世界観へ案内
  - 眺めて見つける(エフェクチュエーション): ギャラリーを自由にめぐる
- **世界観タブ × カルーセル**: バウハウス / アール・デコ / スイス・スタイル / メンフィス / Y2K / ジャパンディ。各世界観に3つのUIモック(コードで再現)+「なぜそう見えるのか」の解説
- **年表**: タブの階層にデザイン史の年表。ノードをクリックすると世界観を横断できる
- **クイズ**: 「このデザイン、どの世界観?」5問。正解すると背景がその世界観に衣装替え
- **背景**: 1枚のぼかし写真のようなアンビエント。選んだ世界観に合わせて色が変わる(=ピエロの衣装替え)。ロゴ「piel●」の「o」はピエロの鼻で、世界観のアクセント色に染まる

## 開発

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # dist/ に本番ビルド
```

Vite + React + Tailwind CSS v4。

## 構成

```
src/
  App.jsx                 # ビュー切替・ヘッダー・背景色の統括
  components/
    Ambient.jsx           # ぼかし背景(世界観ごとに衣装替え)
    Entrance.jsx          # 2つの玄関
    Gallery.jsx           # タブ+カルーセル+「なぜ」解説
    Timeline.jsx          # 年表(全体の位置+世界観のあゆみ)
    Causation.jsx         # 問い形式の診断
    Quiz.jsx              # 何時代クイズ
    MockScaler.jsx        # 560×420キャンバスの等倍縮小
  data/movements.jsx      # 世界観データ(年表・見分けポイント)
  mocks/*.jsx             # 各世界観のUIモック(3点ずつ)
```

新しい世界観を足すには `src/mocks/` にモック3点を作り、`src/data/movements.jsx` に1エントリ追加するだけ。
